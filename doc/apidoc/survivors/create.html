POST /v1/survivors
Add a new survivor to the database

Errors

Code Description
400 The survivor data has validation errors

{

    "age": 30,
    "gender": "male",
    "last_location": "-16.642442,-49.402550",
    "items": [
      {
        "name": "Water",
        "quantity": 10
      },
      {
        "name": "Food",
        "quantity": 10
      },
      {
        "name": "Medication",
        "quantity": 10
      }
    ]
  }
}
