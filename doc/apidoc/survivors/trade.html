PUT /v1/survivors/:id/trade
Trade items with another survivor

Errors

Code Description
404 The survivor was not found
400 There are some validations errors

Examples

PUT /v1/survivors/1/trade
{
  "target_survivor_id": 2,
  "items": {
    "sending": {
      "Water": 1,
      "Medication": 1
    },
    "requesting": {
      "Ammunition": 6
    }
  }
}
200
PUT /v1/survivors/1/trade
{
  "target_survivor_id": ""
}
400
{
  "errors": {
    "target_survivor_id": [
      "can't be blank"
    ]
  }
}
PUT /v1/survivors/-1/trade
404

Params

Param name Description
id
required

Survivor's id

Validations:

  • Must be a Integer
target_survivor_id
required

Survivor's id, who will be the target of the trade

Validations:

  • Must be a String
items
required

Items to be traded

Validations:

  • Must be a Hash
items[sending]
required

Items that will be sent

Validations:

  • Must be a Hash
items[requesting]
required

Items that will be requested

Validations:

  • Must be a Integer
