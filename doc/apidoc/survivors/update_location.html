PUT /v1/survivors/:id/update_location
Update survivor location

Errors

Code Description
404 The survivor was not found
400 The last location has validation errors

Examples

PUT /v1/survivors/1/update_location
{
  "last_location": "-23.681531,-46.875489"
}
200
PUT /v1/survivors/1/update_location
{
  "last_location": ""
}
400
{
  "errors": {
    "last_location": [
      "can't be blank"
    ]
  }
}
PUT /v1/survivors/-1/update_location
404

Params

Param name Description
id
required

Survivor's id

Validations:

  • Must be a Integer
last_location
required

Survivor's last location

Validations:

  • Must be a String
