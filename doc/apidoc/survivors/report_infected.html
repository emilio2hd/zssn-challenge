PUT /v1/survivors/:id/report_infected
Flag a survivor as infected

Errors

Code Description
404 The survivor was not found
400 There are some validations errors

Examples

PUT /v1/survivors/1/report_infected
{
  "survivor_id": 2
}
200
PUT /v1/survivors/1/report_infected
{
  "survivor_id": ""
}
400
{
  "errors": {
    "survivor_id": [
      "Survivor does not exist"
    ]
  }
}
PUT /v1/survivors/-1/report_infected
404

Params

Param name Description
id
required

Survivor's id

Validations:

  • Must be a Integer
survivor_id
required

Survivor's id, who is infected

Validations:

  • Must be a String
